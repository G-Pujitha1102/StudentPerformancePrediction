🎓 Student Performance Prediction 📊

By G. Pujitha

🎯 Objective

To predict students' math scores and identify underperformers using R.

📁 Project Files

  • modeling.R – R script for EDA, modeling, and plots
  • README.md – project documentation

🧪 Methodology

  1. Data preprocessing and cleaning
  2. Model building (Linear Regression, Decision Tree, Random Forest)
  3. Evaluation using RMSE and R²
  4. Visualization using ggplot2

📊 Model Comparison

ModelRMSER²
Linear Regression5.880.87
Decision Tree7.780.74
Random Forest6.460.82

💡 Challenges & Learnings

  • Handling missing/inconsistent data in R
  • Understanding variable impact on performance
  • Data visualization and model evaluation
  • Resolving GitHub merge conflicts

🚀 Run Instructions

Clone the repo, open modeling.R in RStudio, and run the script step-by-step.
